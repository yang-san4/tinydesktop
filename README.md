# TinyDesktop

ブラウザ上で動くレトロ風デスクトップ OS シミュレーター。HTML5 Canvas + バニラ JavaScript のみで構築されており、外部ライブラリは一切使用していません。

440×330px の仮想スクリーン上に CRT モニター風の外観、BIOS ブートシーケンス、ウィンドウマネージャー、13 種のアプリケーションを再現しています。

## スクリーンショット

> `index.html` をブラウザで開くだけで起動します。

## 機能

### デスクトップシステム

- BIOS 風ブートアニメーション
- ドラッグ＆リサイズ可能なウィンドウマネージャー
- タスクバー（ウィンドウ一覧 + システムクロック）
- スタートメニュー（アプリランチャー + テーマ切替）
- 右クリックコンテキストメニュー
- デスクトップアイコンのドラッグ配置
- デスクトップペット
- トースト通知システム
- サウンド ON/OFF（localStorage で永続化）

### 5 つのテーマ

Vapor / Wafuu / Mokume / Classic / Aqua — CSS カスタムプロパティによるリアルタイム切替（リロード不要）

### 搭載アプリケーション

| アプリ | 種別 | 概要 |
|--------|------|------|
| **GEKKO** | 3D FPS | ポリゴンラスタライザ（WebGL 不使用）、AI 敵、グラップルフック、5 ウェーブ制 |
| **Tetris** | ゲーム | SRS 回転、ゴーストピース、7-bag ランダマイザ |
| **Pinball** | ゲーム | 固定タイムステップ物理、フリッパー＆バンパー |
| **Solitaire** | カードゲーム | クロンダイクソリティア、ピクセルアートカード描画 |
| **Minesweeper** | ゲーム | 9×9 グリッド、フラッドフィル探索 |
| **3D Maze** | FPS | レイキャスティング、DFS 迷路生成 |
| **Terminal** | シェル | 27 種の Unix コマンド、パイプ・リダイレクト・変数展開対応 |
| **TinyWeb** | ブラウザ | フェイクウェブブラウザ、ピクセルアートページ |
| **Piano** | 楽器 | Web Audio API、13 鍵クロマチック（C4〜C5） |
| **Aquarium** | ウィジェット | アニメーション魚、テーマ連動 |
| **Clock** | ウィジェット | アナログ＋デジタル時計 |
| **Calendar** | ウィジェット | 月送りカレンダー |
| **Hourglass** | ウィジェット | 砂時計タイマー |

## 使い方

```bash
# ローカルで開く（サーバー不要）
open tinydesktop/index.html

# または任意の HTTP サーバーで配信
npx serve tinydesktop
```

### キーボード操作

**GEKKO（FPS）**
- `WASD` — 移動
- マウス — 視点操作（Pointer Lock）
- クリック — 射撃
- `Space` — ジャンプ（二段ジャンプ対応）
- `Shift` — ダッシュ
- `E` — グラップルフック
- `1` `2` `3` — 武器切替

**Tetris**
- `←` `→` — 左右移動
- `↓` — ソフトドロップ
- `↑` / `X` — 右回転
- `Z` — 左回転
- `Space` — ハードドロップ

**Piano**
- `Z X C V B N M` — 白鍵（C D E F G A B）
- `S D` `G H J` — 黒鍵

**Terminal**
- `ls`, `cd`, `cat`, `grep`, `echo`, `mkdir`, `rm` など 27 コマンド対応
- パイプ `|`、リダイレクト `>` `>>`、`&&` チェーン、`$HOME` 変数展開

## 技術的な特徴

- **ソフトウェア 3D ラスタライザ** — GEKKO は WebGL を使わず Canvas 2D だけでポリゴン描画・Z バッファ・ライティングを実装
- **レイキャスティングエンジン** — 3D Maze は Wolfenstein 方式のレイキャスティングで擬似 3D を実現
- **固定タイムステップ物理** — Pinball はフレームレート非依存の安定したシミュレーション
- **疑似シェル** — Terminal はトークナイザ・パイプ・リダイレクト・ファイルシステムを約 1,400 行で実装
- **AI パスファインディング** — GEKKO の敵 AI はグリッドベース A* + 遮蔽物利用の戦術行動
- **テーマシステム** — CSS カスタムプロパティで全アプリがリアルタイムにテーマ追従

## アーキテクチャ

各アプリは **IIFE（即時実行関数式）** で独立したスコープに隔離されています。グローバル名前空間の汚染を防ぎ、アプリ間の干渉を排除しています。

ウィンドウの開閉・最小化は `MutationObserver` で監視し、非表示時にゲームループを自動停止することでリソースを節約しています。

すべての描画は Canvas 2D API で行い、`image-rendering: pixelated` によるレトロなピクセルアート表現を実現しています。
