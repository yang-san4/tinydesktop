# Aquarium ウィジェット品質アップデート

2026年6月10日。デスクトップの Aquarium ウィジェット（js/aquarium.js、296 → 約560行）を作り直した。世界観の柱である「テーマ連動パレット」「100×60 ピクセルキャンバス」「8fps のレトロなチョッピー感」は一切変えず、水槽の中身を生態系として作り込む方針。

## 魚に種類と生態

- **体型3種** — 丸型（腹のハイライト + 2フレームで開閉する尾びれ）、細身（白い側線ストライプ）、エンゼル型（三角ボディ + 半透明の長い後びれ）
- **行動** — たまに向きを変える / 自発的にダッシュする（白い航跡付き）。画面端ワープだけだった v1 から、水槽の中で「暮らしている」動きに
- **小魚の群れ** — 5匹のミノーが1ユニットでサイン波に散らばって泳ぐ。テーマの2色目を使用
- **底もの** — ナマズ。砂すれすれを低速で進み、たまに立ち止まる。ヒゲと尾を別描画

## 給餌イベント（目玉）

約45秒ごとに餌ペレットが水面から沈んでくる。**魚は最寄りのペレットへ向きを変えて2.2倍速でダッシュ**し、近づくと食べる（ペレット消滅）。小魚の群れも餌の方向へ加速する。水槽の前で待っていると「餌の時間」が始まる、を再現。

## 環境の追加

- **コースティクス** — 砂の上を光の帯が3本流れる
- **水面** — きらめきの粒が走り、泡が到達すると**ポンと弾ける**（白いスプラッシュ1フレーム）
- **泡** — 上昇するほど大きくなる。右端の**エアストーン**からはクラスタで連続発生
- **二枚貝** — 約20秒周期で開き、中の真珠が明滅、開く瞬間に泡をひとつ吐く
- **岩 + サンゴ** — テーマのアクセント色1色目をサンゴに流用（パレット構造は不変更）
- **広葉プラント** — 4枚の葉が扇状に開いてそれぞれ揺れる
- **前面ガラスのカタツムリ** — 約1分ごとに登場し、半透明でガラスを登っていく（最前面に描画 = ガラスに張り付いて見える）

## 実装メモ

- パレット構造（water/sand/fish[7]/bubble/ray...）は5テーマとも無変更。新規要素は既存色の流用（サンゴ= fish[0]、ナマズ= fish[6]、カタツムリ= fish[3]/[5]）なので、テーマを切り替えても世界observが崩れない
- 検証はヘッドレス rAF スタブで約11分相当（5,300フレーム）連続実行 + 5テーマ切替、エラーゼロ
- 8fps ゲートはそのまま。1フレームの描画コストは増えたが 100×60 の fillRect なので無視できる

**学習ポイント**: ウィジェットの「眺める楽しさ」は周期の異なるイベントの重ね合わせで作れる（尾びれ 0.25秒 / 貝 20秒 / 給餌 45秒 / カタツムリ 1分）。次に何が起きるか分からない時間構造が、小さい画面でも飽きさせない。
