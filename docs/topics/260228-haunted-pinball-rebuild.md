# 2026-02-28: Haunted Pinball — ホラーハウステーマ全面リビルド

## 概要
ピンボールゲームを「お化け屋敷」テーマで全面書き換え。物理パラメータ修正、新ギミック追加、ビジュアル・サウンドのホラー化。

## 追加変更 (2回目)

### クリプトホール ループバグ修正
射出後にボールがすぐ再キャプチャされるバグを修正。`CRYPT.cooldown = 90`（1.5秒の無敵時間）を射出時にセットし、キャプチャ条件で `cooldown <= 0` をチェック。

### 振り子時計（Pendulum Clock）追加
テーブル中央 (40, 85) に揺れる大振り子を配置。

```javascript
var PENDULUM = {
  pivotX: 40, pivotY: 85,  // 時計文字盤の位置
  len: 14,                  // 振り子アーム長
  bobR: 3.5,                // ボブ半径
  hitCount: 0               // ヒットカウント（5回でボーナス）
};
```

**ギミック**:
- 物理シミュレーションで常時揺れる振り子（単振子の運動方程式）
- ボールが当たると振り子の動き方向に弾かれる（150pt）
- 5回ヒットで「MIDNIGHT BONUS」発動（+2000pt、紫パーティクル爆発）
- 時計文字盤に時針・分針が回転、ヒットカウントを赤ドットで表示
- ボブに「邪眼」モチーフ（赤い目）

**学習ポイント**: 振り子の物理は `angVel -= gravity * sin(angle)` で実装。振幅が減衰しすぎないよう、低振幅時にエネルギーを補充。ボブの速度 `cos(angle) * angVel * len` をボールに加算することで、振り子の動きとボールの弾かれ方が連動する。

```javascript

// ACTIVE = 閉じた状態（上向き）→ ボールを打ち上げる
```

## 2. 新ギミック

### A. クリプトホール（捕獲穴）
```javascript
var CRYPT = {
  x: 40, y: 132, r: 5,
  state: 'idle', // idle → capturing → holding → ejecting
  timer: 0
};
```
- ボールが穴に近づくと吸い込み → 1.5秒ホールド → ランダムボーナス → 射出
- ボーナス: 500pt / 1000pt / エクストラボール / ターゲットリセット

**学習ポイント**: 4状態のステートマシンで管理。`holding`中はボール物理をスキップし、位置を穴中心に固定。

### B. キックバック
```javascript
var KICKBACKS = [
  { x: 5, y: 30, active: true, flash: 0 },  // 左アウトレーン
  { x: 61, y: 30, active: true, flash: 0 }   // 右アウトレーン
];
```
- 各ボールにつき各サイド1回使用可能
- 棺桶ターゲット全倒しでリチャージ

### C. 墓石バウンサー
```javascript
var TOMBSTONES = [
  { x: 20, y: 98, w: 6, h: 8, points: 50, flash: 0 },
  { x: 60, y: 98, w: 6, h: 8, points: 50, flash: 0 }
];
```
- AABB衝突判定で矩形障害物として機能

## 4. サウンド変更

全サウンドを低音寄り・不気味な音色に変更:
- バンパー: 200→400Hz（旧600→1200Hz）+ 低ノイズ
- フリッパー: 150Hz square（旧200Hz）
- 発射: 100→500Hz sawtooth（旧150→800Hz）
- ドレイン: 400→50Hz 400ms（旧500→80Hz 300ms、「地獄に落ちる音」）
- ボーナス: C→Eb→Gb→C 下降マイナー（旧上昇メジャー）

新規サウンド:
- `playCrypt()`: 100Hz + 150Hz 不協和音
- `playEject()`: 300→800Hz + ノイズ
- `playKickback()`: 250→500Hz square
- `playThunder()`: ノイズバースト + 40Hz ランブル
