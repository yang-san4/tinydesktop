### 2026-06-10

#### GEKKO v2 作り直し
- GEKKO 設計思想を docs から回収: 空中都市アリーナ、垂直機動（ジェット/グラップル/二段ジャンプ/ダッシュ）、5敵タイプ、ウェーブ制
- 過去の教訓: 描画と衝突の二重管理が「見えるのに乗れない」バグの温床 → v2 では一元化必須
- 技術方針: WebGL 低解像度（400×300）+ pixelated 拡大。キャンバス3層（WebGL / 2D HUD / DOM overlay）、シェーダ3本（world/blend/sky）
- ビジュアル: 夕暮れシンセウェーブ空、3D 敵メッシュ（羽ばたき・歩行）、動的ライト4灯、ブロブシャドウ、遠景スカイライン、アトラクトモード
- ゲームフィール: コヨーテタイム、ヒットストップ、FOV キック、ヒット/キルマーカー、コンボ、RAIL ヒットスキャン化。v1 教訓（jumpCD/recentJet/bestTop/アンチスタック）を全移植
- 音: SFX 拡充 + 104BPM の BGM シーケンサ（サウンドトグル連動）
- フォローアップ: v1 を比較用に復元（履歴 abbe613 → js/fps-v1.js、fps-* → fpsv1-* に一括改名）。AI テストモードの ESC デッドロックを manualPause フラグで修正。落下ダメージ 10→20 + コンボ消失 + 0.9秒暗転テレポート復帰 + スピードライン/風切り音演出
- changelog 作成、コミット（8d0bc65 / 990afcb / 1dee36e）
